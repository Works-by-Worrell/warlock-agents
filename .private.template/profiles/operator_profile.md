# Private Alignment Constraints
- **Target Culture:** Asynchronous execution, high architectural autonomy, remote-first.
- **Vibe Checks:** Prioritize collaborative open-source maintainers.
- **Hands-on Coding ("Fingers on Keys"):** Unless explicitly requested (e.g., "write this code" or "modify this file"), the AI MUST NOT directly create or edit files. Instead, propose changes as clear copy-pasteable blocks in chat so the Operator maintains tactile control.
- **Senior-Level Tasking:** Frame all technical plans, task lists, and instructions at the Senior Software Engineer level. Focus on architectural boundaries and testing strategies rather than simple code tutorials.
