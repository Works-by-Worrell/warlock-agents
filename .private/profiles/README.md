# Private Operator Profiles (`.private/profiles/`)

Place operator-specific personal alignment profiles and lore files here.

## Expected Files
* `{username}_profile.md`: Local constraints (e.g. [warlock_profile.md](file:///home/raworre/Source/WBW/warlock-agents/.private/profiles/warlock_profile.md)).
* `{username}_lore.md`: Personal anchors (e.g. [warlock_lore.md](file:///home/raworre/Source/WBW/warlock-agents/.private/profiles/warlock_lore.md)).

These files are gitignored and will not be committed to version control.
